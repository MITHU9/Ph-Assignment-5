What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a hierarchical tree of objects, where each part of the document (like elements, attributes, and text) is represented as a node. The DOM allows programming languages like JavaScript to dynamically access, modify, and manipulate the content, structure, and style of a webpage.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use several built-in methods provided by the document object. Each method allows you to target elements in different ways, such as by their ID, class name, tag name, or CSS selector. Here are the most common ways to select elements from the DOM:
1. document.getElementById()
2. document.querySelector()
3. document.querySelectorAll()
4. document.getElementsByClassName()
5. document.getElementsByTagName()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element instead of attaching individual listeners to multiple child elements. This allows the parent to handle events triggered by its child elements. When an event occurs on a child, it bubbles up to the parent, which can then respond to the event based on the target of the event.
Advantages of Event Delegation
Performance: Attaching a single event listener to a parent instead of multiple listeners to each child element reduces memory usage and improves performance, especially when dealing with a large number of child elements.
Dynamic Content: If child elements are added or removed dynamically (e.g., through AJAX or user actions), you donâ€™t need to reattach event listeners to each new element. The parent listener will handle events for any current or future children.
Simpler Code Management: Managing one event listener is often simpler than managing multiple listeners. This reduces the complexity of your code and can help prevent bugs.
Easier Event Handling: You can handle events based on the specific child element that triggered the event, allowing for more flexible event management.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM (Document Object Model), you can use various JavaScript methods and properties.

1. Manipulating Attributes

getAttribute()
This method is used to retrieve the value of an attribute.
let value = element.getAttribute("attributeName"); let link = document.getElementById("myLink");
let hrefValue = link.getAttribute("href");
console.log(hrefValue);

settAttribute()
This method is used to set or change the value of an attribute.
element.setAttribute("attributeName", "value"); let img = document.getElementById("myImage");
img.setAttribute("src", "new-image.jpg");

Direct Property Manipulation
Some attributes can be manipulated directly via element properties.
element.id = "newId";
element.className = "newClass"; let input = document.getElementById("myInput");
input.value = "New value";

2. Manipulating Styles

style Property
You can modify an element's inline styles using the style property. This gives access to specific CSS properties in camelCase format.
element.style.propertyName = "value"; let div = document.getElementById("myDiv");
div.style.backgroundColor = "blue";
div.style.width = "100px";
div.style.height = "100px";

classList Property
classList is a built-in property in JavaScript that provides access to the list of classes on an element. element.classList.add("newClass");
element.classList.remove("oldClass");
element.classList.toggle("newClass");
let hasClass = element.classList.contains("newClass");
